# GPO---Verifica-di-laboratorio-Nardullo-Pinna
Benvenuti nel repository ufficiale del progetto per la verifica di laboratorio GPO 2.
Questo progetto implementa un sistema di gestione attività (TaskMaster Pro) sviluppato simulando un workflow aziendale moderno.

 
🚀 Il Progetto
 
L'applicazione permette agli utenti di gestire una lista di cose da fare direttamente dal browser. È stata estesa partendo da una base semplice con due feature principali.

 
🛠 Tecnologie
 
HTML5 / CSS3
JavaScript
